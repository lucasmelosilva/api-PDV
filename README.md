# API PDV
It is a just simple project of the POS api.

## Installation and Run

### Installation
```bash 
git clone https://github.com/lucasmelosilva/api-PDV
```

### Run

```bash
npm run dev
```
## API

### /api/products

* `POST` : Add new Product

### /api/products/load
* `POST` : Load a product by bar code
