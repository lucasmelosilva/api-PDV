# API PDV
It is a just simple project of the POS api.

## Getting started

### Install
```bash 
$ git clone https://github.com/lucasmelosilva/api-PDV
$ cd ./api-PDV
```
### NPM
```bash
$ npm install
$ npm run dev
```
## API

### /api/products

* `POST` : Add new Product

### /api/products/:barcode
* `GET` : Load a product by bar code
